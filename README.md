# Book Tracker

Автор: Линников Алексей

## Описание программы

Это графическое приложение для учета прочитанных книг. Можно добавлять книги с информацией о названии, авторе, жанре и количестве страниц. Есть возможность фильтровать список по жанру и количеству страниц, а также сохранять и загружать данные в JSON файлы.

## Как запустить

1. Убедитесь, что у вас установлен Python 3.
2. Скачайте или скопируйте файлы проекта.
3. Запустите `book_tracker.py` двойным кликом или через терминал командой:

```bash
python path/to/book_tracker.py
